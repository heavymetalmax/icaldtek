## 🚀 Як розповсюджувати

### 1. На GitHub

```bash
# Створіть репозиторій на github.com/new
git remote add origin https://github.com/ВАШ_НІКНЕЙМ/dtek-calendar
git branch -M main
git push -u origin main
```

### 2. Включіть Actions

1. Settings → Actions → General
2. "Allow all actions and reusable workflows"
3. Save

### 3. Готово!

**Календар буде оновлюватися автоматично на GitHub.**

## 📊 Що відбувається

### Workflow: `update.yml`
- ⏰ Запускається кожні 15 хвилин
- 🌐 Отримує дані з ДТЕК
- 📅 Генерує календар
- 💾 Комітить оновлення
- 📤 Зберігає артефакт

### Workflow: `generate-custom.yml`
- 🎯 Запускається вручну
- 📍 Користувач вводить адресу через GitHub UI
- 📅 Генерується календар для цієї адреси
- 📥 Завантажується як артефакт

---

## 👥 Для користувачів

1. **Fork репозиторію**
2. **Settings → Actions → Дозволити**
3. **Змінити config.json на свою адресу**
4. **Commit & Push**
5. **Календар оновлюватиметься автоматично!**
